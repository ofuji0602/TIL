File・IO 1-12
カレントフォルダにファイル「test.txt」が存在しないとき、「ファイルが見つかりませんでした」と出力するコードを1つ選択してください。

1.  

if File.nil?("test.txt")
  puts "ファイルが見つかりませんでした" 
end
2.  

if File("test.txt").nil?
  puts "ファイルが見つかりませんでした" 
end
3.  

unless File.exist?("test.txt")
  puts "ファイルが見つかりませんでした" 
end
4.  

unless File("test.txt").exist?
  puts "ファイルが見つかりませんでした" 
end
